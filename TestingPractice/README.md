# Testing Practice

This repository is a JavaScript practice project focused on building and testing small utility functions.

It includes simple exercises for working with strings, arrays, and basic math, along with Jest tests to verify the results.

## What it includes

- `capitalize(string)` - capitalizes the first letter of a string
- `reverseString(string)` - reverses a string
- `calculator` - basic arithmetic methods
- `caesarCipher(word, shift)` - shifts letters in a string
- `analyzeArray(numberedArray)` - returns average, min, max, and length

## Install

```bash
npm install
```

## Test

```bash
npm test
```

## Files

- `index.js` - source functions
- `index.test.js` - Jest tests
